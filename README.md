# GenAI Text Generator (Google Gemini)

This project demonstrates a simple Generative AI application built using
Python and Google Generative AI (Gemini). The project was created after
completing Google's *Introduction to Generative AI* certification.

## Features
- Interactive command-line Generative AI
- Uses Google Gemini API
- Secure API key handling using environment variables

## Technologies Used
- Python
- Google Generative AI (Gemini)
- Prompt Engineering

## How to Run
1. Install dependencies  
   `pip install -r requirements.txt`

2. Set API key as environment variable  
   - Windows (PowerShell):
     ```
     setx GEMINI_API_KEY "your_api_key_here"
     ```
   - macOS/Linux:
     ```
     export GEMINI_API_KEY="your_api_key_here"
     ```

3. Run the application  
   `python genai_app.py`
